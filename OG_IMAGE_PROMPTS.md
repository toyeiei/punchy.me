## Generation Tips

### For Midjourney:
1. Use these exact prompts in Discord
2. Add `--ar 1200:630` for correct aspect ratio
3. Use `--style raw` for photorealistic results
4. Upscale your favorite result (U1, U2, U3, U4)
5. Download and convert to WebP format

### For DALL-E (ChatGPT):
1. Use the "Alternative Simpler Prompt" versions
2. Request 1200x630 dimensions explicitly
3. May need to specify "no text overlays" if it adds words
4. Download and convert to WebP

### For Other AI Tools (Leonardo, Stable Diffusion):
- Use the full prompts but remove Midjourney-specific flags (--ar, --style, --v)
- Adjust aspect ratio settings in the tool's UI
- Emphasize "1200x630 resolution" in prompt
